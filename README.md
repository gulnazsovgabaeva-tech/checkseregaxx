# Бот-уведомлятель о скидках на пижамы Victoria's Secret

Следит за страницей https://www.victoriassecret.com/us/vs/sleepwear/pajama-sets
и присылает сообщение в Telegram, когда появляются новые акции
(проценты, "Buy 1 Get 1", "Buy 2", "Save $..." и т.п.).

Работает бесплатно в облаке через GitHub Actions. Компьютер держать включённым не нужно.

## Файлы
- `check_sales.py` — основной скрипт
- `requirements.txt` — зависимости
- `state.json` — память бота (что он видел в прошлый раз)
- `.github/workflows/check.yml` — расписание запуска (каждые 30 минут)

## Что нужно настроить
1. Создать бота в Telegram через @BotFather и получить ТОКЕН.
2. Узнать свой CHAT_ID через @userinfobot.
3. Создать публичный репозиторий на GitHub и загрузить эти файлы
   (файл check.yml положить по пути `.github/workflows/check.yml`).
4. В настройках репозитория добавить два секрета:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
5. Во вкладке Actions запустить workflow вручную для проверки.

Подробная пошаговая инструкция — в сообщении от ассистента в чате.

## Как поменять частоту проверки
В файле `check.yml` строка `cron: "*/30 * * * *"`:
- каждый час: `"0 * * * *"`
- каждые 15 минут: `"*/15 * * * *"`

## Если перестало работать
Если сообщения вообще не приходят даже после ручного запуска — возможно, сайт
заблокировал автоматический заход. Тогда нужно усложнить способ загрузки страницы
(добавить прокси или другой метод). Напиши об этом ассистенту.
